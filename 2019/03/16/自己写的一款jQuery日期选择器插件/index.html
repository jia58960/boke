基于jQuery写的一款日期选择器插件

兼容性

IE 6/7/8/9/10/11、Firefox、chrome、Safari、360、搜狗…

核心功能

  • 选择日期，翻月，翻年，选择年份区间。
  • 提供农历信息。
  • 特色功能：强容错、按键控制（支持up、down、left、right、Enter、Esc、pageUp、pageDown）、不限于绑定元素、自动定位弹框位置（自动判断上下左右距离选择合适位置弹框）、回调函数（自行处理返回的日期对象）
  • 代码特色：模板数据分离、模板引擎、DOM节点缓存、按需加载…
